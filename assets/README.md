# Site assets

## Social share image

| File | Purpose |
|------|---------|
| `og-default.jpg` | Default Open Graph / Twitter share image (1200×630). Captured from the homepage hero with the lantern vertically aligned to the “Great Lantern Festival” title. The SEO build script emits `og:image` and `twitter:image` when this file exists. |
| `og-home-mobile.png` | Raw mobile viewport capture of the homepage (reference source). |

To refresh after hero visual changes: capture the homepage hero with title + lantern in frame, then compose into `og-default.jpg` at 1200×630.
